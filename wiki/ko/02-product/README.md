# Product 개요

Config 제품을 **Data Platform**과 **Foundation Model** 두 축으로 정리한 개요입니다.

---

## 미션 맥락

> We build the data infrastructure and technology that enable **general-purpose bimanipulation**.

- 데이터 무결성은 **closing the loop**(학습 → 물리 세계 검증)으로 확보
- 고객과 협력해 데이터가 실제 배포 성공의 촉매가 되도록 함
