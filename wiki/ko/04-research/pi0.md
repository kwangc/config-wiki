## 2) 상세 내용

### 핵심 아이디어: 사전학습 VLM 위에 올린 플로우 매칭 액션 전문가

RT-2, OpenVLA 같은 기존 VLA는 로봇 액션을 토큰으로 이산화한 뒤 언어 모델이 자기회귀적으로 예측하는 방식을 사용한다. LLM 디코딩 파이프라인을 재활용할 수 있어 편리하지만, 양자화 오류가 생기고 샘플링이 느리며, 정교하고 접촉이 많은 조작에 요구되는 부드러운 움직임을 만들어내기 어렵다.

π0는 이 문제를 깔끔한 아키텍처 분리로 해결한다. **사전학습 VLM은 인식과 언어 이해를 담당**하고, **별도의 플로우 매칭 액션 전문가가 연속적인 액션 생성을 담당**한다. VLM 백본은 PaliGemma(3B 파라미터)로, 이미지 관측과 자연어 태스크 지시를 처리한다. 액션 전문가는 소형 트랜스포머(~300M 파라미터)로, VLM의 토큰 표현을 크로스-어텐션으로 받아 가우시안 노이즈에서 출발해 학습된 ODE(상미분방정식) 적분을 반복하며 액션을 생성한다. 이것이 플로우 매칭의 핵심 메커니즘이다.

### 아키텍처

- **VLM 백본 (PaliGemma, 3B):** 하나 또는 여러 카메라 이미지와 자연어 지시를 함께 처리한다. 시각적 장면 이해와 언어 그라운딩이 모두 담긴 풍부한 문맥 토큰 임베딩 시퀀스를 출력한다.
- **액션 전문가 (~300M 트랜스포머):** VLM 임베딩을 크로스-어텐션으로 조건으로 받는 전용 트랜스포머다. 추론 시 액션 공간의 가우시안 노이즈에서 출발해 ~10번의 학습된 ODE 적분 스텝으로 깨끗하고 연속적인 액션에 도달한다.
- **액션 표현:** 이산화 없이 원시 연속 값(관절 각도 또는 엔드이펙터 델타)을 사용한다. 짧은 예측 호라이즌에 해당하는 전체 액션 청크를 토큰 하나씩이 아닌 한 번에 생성한다.
- **크로스-어텐션 브리지:** 액션 전문가는 노이즈 제거 스텝마다 VLM 토큰 표현 전체에 어텐션을 수행하므로, 고수준 의미 맥락이 저수준 운동 출력을 전 과정에 걸쳐 안내한다.

### 플로우 매칭이 디퓨전보다 나은 이유

플로우 매칭과 디퓨전 폴리시는 모두 노이즈를 액션으로 변환하는 방법을 학습해 액션 분포를 모델링하지만, 학습·추론에서 사용하는 확률 경로가 다르다:

- **디퓨전 (DDPM):** 확률적 미분방정식(SDE)을 따르며 굴곡진 노이즈 경로를 사용해 추론 시 ~100번의 노이즈 제거 스텝이 필요하다.
- **플로우 매칭:** 노이즈와 데이터 사이의 직선 ODE 경로를 학습한다. 경로가 직선에 가까울수록 더 적은 적분 스텝(~10번)으로 좋은 액션에 도달할 수 있어 추론 지연을 약 10배 단축한다. 각 스텝의 회귀 목표가 단순(굴곡진 스코어 대신 직선 방향)하므로 학습도 더 안정적이다.

50 Hz 이상의 고빈도 제어가 필요한 실시간 로봇 제어에서 이 지연 단축은 실질적으로 중요한 의미를 가진다.

### 학습

π0는 파운데이션 모델 패러다임을 따라 두 단계로 학습된다:

1. **다양한 데이터로 사전학습:** Physical Intelligence는 단일팔·바이매뉴얼·모바일 조작 등 다양한 로봇 형태와 일상적인 조작 태스크를 포괄하는 대규모 내부 데모 데이터셋을 구축했다. 모델은 이 혼합 데이터로 사전학습해 범용 감각운동 능력과 언어 그라운딩을 획득한다. 이 데이터셋은 공개되지 않는다.
2. **태스크별 파인튜닝:** 사전학습 모델을 각 목표 태스크의 데모로 모방 학습을 통해 파인튜닝한다. 백본에 이미 강한 사전 지식이 담겨 있으므로 비교적 적은 태스크별 데모로도 수렴한다.

VLM 백본의 언어 조건부 처리 덕분에 재학습 없이 다르게 표현된 태스크 설명이나 새로운 지시에 제로샷으로 일반화할 수 있다.

### 결과

π0는 정밀하고 접촉이 많은 운동 동작을 요구하는 정교한 조작 태스크 스위트에서 평가된다:

| 태스크 | 설명 | 주요 결과 |
|--------|------|---------|
| 세탁물 접기 | 구조화되지 않은 의류를 바이매뉴얼로 접기 | 이전 VLA 기준 모델 대비 우수; 새로운 의류 유형에도 일반화 |
| 테이블 치우기 | 어질러진 테이블 위를 정리하고 닦기 | 물체 배치 변화에 강한 일반화 |
| 박스 조립 | 두 손의 협응 접촉이 필요한 골판지 박스 조립 | 인간 수준에 근접하는 완료율 |
| 전체 조작 | 다양한 조작 스킬 통합 결과 | 복잡한 장기 태스크에서 이전 SOTA(RT-2, OpenVLA) 대비 약 2배 향상 |

논문은 π0가 정교함과 다단계 추론이 요구되는 태스크에서 RT-2, OpenVLA 모두를 크게 능가하면서, 단순한 집기-놓기 태스크에서는 경쟁력 있는 성능을 유지한다고 보고한다.

### 한계

- **비공개 모델과 데이터:** 사전학습 데이터셋과 모델 가중치가 공개되지 않아 독립적인 재현이나 커뮤니티 파인튜닝이 불가능하다.
- **하드웨어 의존성:** 평가가 Physical Intelligence의 독자적인 로봇 하드웨어에서 이뤄져 다른 형태의 로봇으로 결과가 그대로 전이되지 않을 수 있다.
- **공개 벤치마크 없음:** OpenVLA(BridgeV2 수치 보고)와 달리 π0 평가는 내부 태스크 스위트에서 이뤄져 다른 발표된 방법들과 직접적인 비교가 어렵다.
- **요구 데이터 규모:** 파운데이션 모델 사전학습은 대부분의 조직이 쉽게 복제하기 어려운 대규모 독점 데이터셋에 의존한다.

## 3) 왜 중요한 논문인가

- **플로우 매칭이 로봇 폴리시 학습에 진입했다.** π0는 플로우 매칭이 — 디퓨전이나 자기회귀 토큰 예측 대신 — 정교한 조작을 위한 올바른 액션 생성 메커니즘임을 대규모로 처음 입증했다. 더 빠른 추론(~10스텝), 부드러운 액션 궤적, 더 안정적인 학습의 조합은 이를 미래 로봇 폴리시 아키텍처의 유력한 기본값으로 만든다.
- **추론과 행동의 분리가 아키텍처적으로 우월하다.** 기존 VLA는 인식, 언어 그라운딩, 액션 생성을 단일 자기회귀 모델에 통합했다. π0는 VLM과 액션 전문가를 분리하고 크로스-어텐션으로만 연결하면 정밀 태스크에서 더 나은 성능을 달성하면서도 VLM의 광범위한 언어·시각 이해를 유지할 수 있음을 보여준다.
- **파운데이션 모델 사전학습이 정교한 조작에서도 유효하다.** 대규모 사전학습이 접촉이 많은 태스크(대략적인 집기-놓기 이상)에도 의미 있게 전이될 수 있는지는 열린 질문이었다. 세탁물 접기와 박스 조립에서 π0가 거둔 성공은 그것이 가능함을 강하게 시사하는 증거다.
- **범용 로봇 폴리시의 새로운 능력 기준을 세웠다.** 세탁물 접기와 조립 결과는 이전의 범용 폴리시들이 안정적으로 달성하던 수준보다 질적으로 한 단계 높은 성과를 보여주며, 파운데이션 모델 접근법이 복잡한 태스크에서 이제 태스크 특화 폴리시와 경쟁하거나 능가할 수 있음을 알린다.

## 4) Config 적용 사례

- **바이매뉴얼 액션에 플로우 매칭 적용:** Config의 바이매뉴얼 조작은 고차원의 연속 액션 공간을 다룬다. 플로우 매칭은 토큰 기반 예측보다 더 자연스러운 선택이다 — 관절 각도의 양자화를 피하고, 부드러운 궤적을 생성하며, 고빈도 제어 루프에 충분히 빠르게 실행된다.
- **VLM + 액션 전문가 분리:** Config의 아키텍처는 인식·언어 추론과 운동 액션 생성을 크로스-어텐션으로 연결된 별도 모듈로 분리해야 한다. LLM의 토큰 어휘를 통해 액션을 강제하는 대신 이 분리를 채택하면 각 구성 요소를 독립적으로 최적화하고 다른 쪽을 재학습하지 않고도 업그레이드할 수 있다.
- **파운데이션 → 파인튜닝 패러다임:** π0의 학습 레시피 — 다양한 멀티 형태 사전학습 후 태스크별 파인튜닝 — 는 Config의 데이터 전략에 직접적인 템플릿을 제공한다. 초기 데이터 수집에서는 폭넓은 다양성(다양한 태스크, 다양한 물체, 다양한 환경)을 우선하고, 이후 배포 목표에 맞게 특화한다.
- **정교함 벤치마크:** π0의 세탁물 접기와 박스 조립 태스크는 Config가 목표로 하는 접촉이 많은 바이매뉴얼 조작 과제와 유사한 도전이다. π0의 성능을 달성하거나 능가해야 할 능력 기준으로 삼으면 내부 구성 기준 대신 외부 검증된 구체적인 벤치마크를 갖게 된다.
