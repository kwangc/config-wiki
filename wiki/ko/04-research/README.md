# Research — 논문·기술 리뷰 인덱스

논문, 기술 블로그, 아티클 요약을 모아두는 곳.  
각 항목은 한 줄 요약 + 핵심 포인트 + 링크 형태로 정리하면 나중에 검색·참조하기 좋습니다.

---

## 작성 템플릿 (예시)

```markdown
### [논문/글 제목](URL)
- **한 줄:** ...
- **핵심:** ...
- **키워드:** #VLA #bimanual #data
- **날짜:** YYYY-MM
```
