# Glossary

로봇·VLA·데이터 인프라 관련 **핵심 용어** 한 줄 정의 모음. (채워나가기)

---

## 로봇·제어

| 용어 | 정의 |
|------|------|
| **Bimanual** | 두 손(듀얼 암/엔드이펙터) 협응으로 수행하는 조작 |
| **Teleoperation (teleop)** | 사람이 원격으로 로봇을 조작해 데모를 수집하는 방식 |
| **Proprioception** | 로봇 자신의 관절·위치·속도 등 상태 감각 |
| **Manipulation** | 그랩, 플레이스, 조립 등 물체를 다루는 행위 |

---

## 시뮬레이션·전이

| 용어 | 정의 |
|------|------|
| **Sim2Real** | 시뮬레이션에서 학습한 정책/모델을 실세계 로봇에 전이하는 것 |
| **Reality gap / Domain gap** | 시뮬과 현실 간 차이(물리·센서·외관 등)로 인한 성능 저하 |
| **Domain randomization** | 시뮬 파라미터(광원, 질감, 마찰 등)를 랜덤화해 전이 성능을 높이는 기법 |

---

## 모델·학습

| 용어 | 정의 |
|------|------|
| **VLA** | Vision-Language-Action — 비전·언어·액션을 한 모델로 다루는 로봇 정책 |
| **Foundation model** | 넓은 태스크/데이터로 사전학습한 뒤 특정 태스크로 파인튜닝하는 모델 |
| **Closing the loop** | 학습 → 실세계 테스트 → 데이터/모델 개선을 반복해 검증하는 과정 |

---

## 배포·인프라

| 용어 | 정의 |
|------|------|
| **On-device deployment** | 모델을 로봇/엣지 디바이스에 올려 클라우드 없이 추론하는 방식 |
| **Quantization** | 가중치/활성화를 저정밀도(예: 4bit, 8bit)로 줄여 속도·메모리 절약 |
