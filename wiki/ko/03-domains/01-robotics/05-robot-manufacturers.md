# 로봇 하드웨어 레퍼런스

양팔 로보틱스 연구 및 데이터 수집에 관련된 로봇 암, 이동형 매니퓰레이터, 그리퍼 카탈로그. 스펙, ROS/API 지원 여부, 연구 활용 관점 메모 포함.

---

## 로봇 암

### ROAS PiPER (AgileX OEM)

[제품 페이지](https://roas.co.kr/piper/) · 국내 유통: 로아스, 제조: AgileX Robotics

연구 및 이동형 매니퓰레이션을 위한 경량 6자유도 암. 가성비 입문용으로 국내 대학·스타트업 연구실에서 많이 사용.

| 스펙 | 값 |
|------|----|
| 자유도 | 6 |
| 가반하중 | 1.5 kg |
| 도달범위 | 626 mm |
| 반복정밀도 | ±0.1 mm |
| 무게 | 4.2 kg |
| 입력전압 | DC 24V |
| 통신 | CAN |
| 소재 | 알루미늄 합금 + 폴리머 쉘 |
| 동작온도 | −20°C ~ 50°C |

**소프트웨어:** Python API, ROS 1, ROS 2, 드래그 티칭, 오프라인 궤적

**메모:**
- 무게 4.2 kg — 이동형 베이스 탑재 적합
- 가반하중 1.5 kg — 그리퍼 + 물체 조합에 제약
- AgileX LIMO, Tracer 등 이동 베이스와 자연스럽게 연동

### Universal Robots UR 시리즈

[제품 페이지](https://www.universal-robots.com/) · 덴마크 기업, Teradyne 인수(2015)

산업 표준 협동 로봇(cobot). 전 세계에서 가장 많이 쓰이는 연구용 암. 전 모델 6자유도; e-시리즈는 내장 F/T 센서 포함.

| 모델 | 가반하중 | 도달범위 | 무게 | 비고 |
|------|---------|---------|------|------|
| UR3e | 3 kg | 500 mm | 11.2 kg | 소형 테이블탑 |
| UR5e | 5 kg | 850 mm | 20.6 kg | 가장 많이 쓰이는 연구용 |
| UR10e | 10 kg | 1,300 mm | 33.5 kg | 넓은 작업공간 |
| UR16e | 16 kg | 900 mm | 33.5 kg | 고가반·컴팩트 |
| UR20 | 20 kg | 1,750 mm | 64 kg | 긴 도달범위 |
| UR30 | 30 kg | 1,300 mm | 63.5 kg | 고중량 |

**소프트웨어:** ROS / ROS 2 (공식 드라이버), Python, URScript, PolyScope UI, 1 kHz 제어루프

**메모:**
- 반복정밀도 ±0.03 mm (e-시리즈) — 협동로봇 최고 수준
- 손목 F/T 센서 내장 (e-시리즈)
- 그리퍼·비전·서드파티 생태계 매우 풍부
- UR5e는 다수 오픈소스 로봇 학습 데이터셋의 표준 암 (RoboAgent 등)

### Franka Robotics FR3

[제품 페이지](https://franka.de/research) · 독일 기업

모든 관절에 토크 센서를 갖춘 7자유도 연구용 암. 접촉 기반 조작 연구의 사실상 표준. ACT, Diffusion Policy 등 주요 논문에서 광범위하게 사용.

| 스펙 | 값 |
|------|----|
| 자유도 | 7 (여유 자유도) |
| 가반하중 | 3 kg |
| 도달범위 | 855 mm |
| 반복정밀도 | < ±0.1 mm |
| 관절 토크 센서 | 전 7개 관절 |
| 제어루프 (FCI) | 1 kHz |

**소프트웨어:** FCI(C++, ROS 2, MATLAB); Desk UI (빠른 프로토타이핑)

**메모:**
- 관절 토크 센서 → 임피던스/힘 제어 및 충돌 감지
- 7자유도 여유 자유도 → 양팔 작업 시 특이점 회피·팔꿈치 재배치 용이
- **FR3 Duo**: 공유 베이스 위 양팔 구성 — 양팔 모방학습 연구용으로 인기
- **Mobile FR3 Duo**: 이동 베이스 + 온보드 컴퓨트·센서 탑재
- OTA 업데이트 지속 제공 (System Image 5.x 시리즈)
- 독일산 수출품 → 일부 지역 조달 리드타임 유의

## 이동형 로봇 (with 로봇 암)

### Rainbow Robotics RB-Y1

[제품 페이지](https://www.rainbow-robotics.com/rby1) · 레인보우로보틱스 (코스닥 상장)

6자유도 단일 다리형 토르소를 갖춘 바퀴형 양팔 로봇. 생활 공간에서의 범용 조작을 목표로 설계.

| 스펙 | 값 |
|------|----|
| 총 자유도 | 24 |
| 팔 | 7 DOF × 2 |
| 다리(토르소) | 6 DOF |
| 그리퍼 | 1 DOF × 2 |
| 바퀴 | 1 DOF × 2 |
| 팔 가반하중 | 3 kg (한팔 당) |
| 전체 크기 | 600 × 690 × 1,400 mm (W × D × H) |
| 전체 무게 | 131 kg (상체 38 / 하체 42 / 모바일 51) |

**소프트웨어:** 웹 매뉴얼, GitHub SDK, ROS 지원

**메모:**
- 6자유도 단일 다리 토르소 → 고정 토르소 이동형 대비 수직 도달 범위 우수
- 7자유도 암 + 3 kg 가반하중 → 실제 가정 환경 조작 작업 수행 가능
- 국내 기업 — 국내 연구·스타트업 배포 시 지원 유리
- RB-Y1은 레인보우의 휴머노이드 인접 플랫폼; 별도로 UR 호환 양팔 시스템도 제조

### Dexmate Vega

[제품 페이지](https://www.dexmate.ai/product/vega) · Dexmate Inc., 미국 산타클라라

고자유도 이동형 양팔 로봇. 다지 핸드, 접이식 토르소, 전방향 베이스 탑재. 이동+조작 통합 연구용 범용 플랫폼.

| 스펙 | 값 |
|------|----|
| 총 자유도 | 36+ |
| 암 | 13 DOF × 2 |
| 토르소 | 3 DOF (접이식) |
| 헤드 | 3 DOF |
| 베이스 | 전방향 이동 |
| 팔 가반하중 | 10+ lbs (~4.5 kg) |
| 최대 수직 도달 | 2.2 m (접이식 확장 시) |
| 운용 시간 | 10+ 시간 (무부하 30+ 시간) |
| 컴퓨트 | NVIDIA AGX Orin (Thor 업그레이드 예정) |

**센서:** RGBD+RGB 카메라, LiDAR, IMU, 초음파, 6축 F/T

**소프트웨어:** Python API (`pip install`), URDF/USD 시뮬레이션 지원, VR/외골격 원격조작, 플릿 관리

**메모:**
- 접이식 토르소 — 높은 선반 작업 시 2.2 m 확장, 이동 시 접기 가능
- 13자유도 암 — 7자유도 대비 더 정교하나 시연 데이터 수집 난도 높음
- 현재 양산 중 (리드타임 약 1개월, 2026년 기준)
- 다지 핸드 + F/T 센서 → 접촉 기반 정교 조작에 적합

## 그리퍼

### Robotiq 2F-85 / 2F-140

[제품 페이지](https://robotiq.com/products/2f85-140-adaptive-robot-gripper) · Robotiq (캐나다)

로봇 학습 연구에서 가장 많이 쓰이는 평행 그리퍼. UR·FR3와 폭넓게 호환.

| 스펙 | 2F-85 | 2F-140 |
|------|-------|--------|
| 최대 개구 | 85 mm | 140 mm |
| 가반하중 | 5 kg | 2.5 kg |
| 무게 | 900 g | 1,000 g |

- USB / RS-485 / Ethernet 지원
- OXE, BridgeData 등 주요 오픈소스 데이터셋의 표준 그리퍼
- LG전자 로봇선행연구소 협업 시 사용 (Robotiq 검정 그리퍼)

### Fin Ray / UMI 그리퍼

[UMI 논문](https://arxiv.org/abs/2402.10329) · 오픈소스

UMI(Universal Manipulation Interface) 데이터 수집용 컴플라이언트 2손가락 그리퍼. Fin Ray 효과로 손가락이 물체 형상에 맞게 변형됨.

- 3D 프린팅 손가락 + 커플러
- Robotiq 베이스 커플러와 호환
- 저비용; UMI 논문 이후 광범위하게 복제·사용
- Config도 원격조작 데이터 수집 시 Fin Ray + 커플러 설계 활용

### Allegro Hand

[제품 페이지](https://www.wonikrobotics.com/robot-hand) · 원익로보틱스 (국내)

16자유도 4손가락 정교 핸드. 손내 조작(in-hand manipulation) 연구 표준.

| 스펙 | 값 |
|------|----|
| 자유도 | 16 (4손가락 × 4DOF) |
| 가반하중 | ~0.3 kg |
| 통신 | CAN / EtherCAT |

- ROS 드라이버 지원
- 고자유도 → 원격조작 데이터 수집 난도 상승
- 정교 조작 연구에 활용되나 VLA 학습 파이프라인에는 아직 미일반화
