# 모델 클래스

고수준 모델 군: **VLM**, **LLM**, **VLA**.

---

## 개요

- **VLM** — Vision-Language Model: 이미지/영상 + 텍스트 입력, 텍스트(또는 구조화 출력) 출력.
- **LLM** — Large Language Model: 텍스트 입출력; 추론·지시 따르기 기반.
- **VLA** — Vision-Language-Action: 이미지/영상 + (선택) 언어 입력, **로봇 액션** 출력; 액션 헤드로 VLM/LLM 확장.
